GraphQL Customer Impersonation Token Example

This is a simple app to demonstrate server-side use of a Customer Impersonation Token. An easy way to see this in action is to assign a customer to a customer group that has limited category access, and then use their ID to make the request.

The GraphQL Query will be for a simple list of products.

This is the token you receive from the BigCommerce API.
This is used to set the X-Bc-Customer-Id header in the request.
